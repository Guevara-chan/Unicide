# ⋮Sum⋮
__Unicide__ is an avant-garde text formatter, made to assist pluricellular entites with understands speech patterns from [Tinder](https://tinder.com/), [Badoo](https://badoo.com), [Lenorado Dai Winchic](https://vk.com/dayvinchik) and other similar stuff. Made with frustration for frustration.

# ⋮Featuræ⋮
* Every sentence ends with dot or _one_ other corresponding temrination symbol.
* Every closing bracket should be precended with opening one.
* There are single space before comma, not after.
* First letter of every sentence is capitalized.
